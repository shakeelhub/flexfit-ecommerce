## 🔥 What is FlexFit?

**FlexFit** is a modern, fully responsive eCommerce platform built for premium activewear.  
Clean UI. Smooth UX. Fast as your PR.

Built with **React + Vite + Tailwind CSS** — it's not just a store, it's an experience.  
Every page, every interaction, every transition designed to feel effortless.

## 🚀 Quick Start

### Prerequisites

```
✅ Node.js 18+
✅ npm or yarn
```

### 1️⃣ Clone

```bash
git clone https://github.com/shakeelhub/flexfit-ecommerce.git
cd flexfit-ecommerce
```

### 2️⃣ Install

```bash
npm install
```

### 3️⃣ Run

```bash
npm run dev
# → http://localhost:5173
```

### 4️⃣ Build for Production

```bash
npm run build
```
